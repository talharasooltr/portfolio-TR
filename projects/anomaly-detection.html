Deep Learning Manufacturing

Visual Anomaly Detection for Manufacturing QA

An edge-deployed defect detection system that uses unsupervised deep learning to flag surface anomalies on a production line in real time — requiring only 50 training samples per product SKU and cutting defective shipments by 83%.

Role

ML & Edge Deployment Engineer

Timeline

3.5 months

Stack

Python, PyTorch, PatchCore, NVIDIA Jetson, TensorRT, FastAPI

Problem

The manufacturing client produced 15+ product SKUs with varying surface finishes. Traditional defect detection required large labeled datasets per SKU — impractical for low-volume runs. Manual visual inspection caught only 60% of surface defects, leading to costly returns and customer complaints.

Approach

I designed a two-model architecture optimized for few-shot anomaly detection:

  • CNN Autoencoder: Trained exclusively on defect-free images, the autoencoder learns the "normal" appearance of each SKU. At inference, anomalous regions produce high reconstruction error, creating a pixel-level anomaly heatmap.
  • PatchCore: For products with subtle texture defects, PatchCore provides few-shot detection by building a memory bank of normal patch features from only 50 reference images, then flagging patches that deviate from the learned distribution.
  • Edge Deployment: Both models were optimized with TensorRT and deployed on NVIDIA Jetson devices mounted directly on the production line, running inference at 25 FPS with under 40ms latency.
  • Rejection API: When an anomaly confidence exceeds the threshold, the system triggers a pneumatic rejection arm via a FastAPI endpoint, removing the defective item from the conveyor.

Outcome

The system achieved a 96% defect detection rate with a false positive rate below 2%. Defective shipments dropped by 83% within the first month. The few-shot approach meant new SKUs could be onboarded in under 2 hours with minimal data collection.

Key Technologies

  • CNN Autoencoder — Unsupervised anomaly detection via reconstruction error
  • PatchCore — Few-shot defect detection with memory bank
  • NVIDIA Jetson — Edge inference hardware
  • TensorRT — Model optimization for real-time inference
  • FastAPI — Rejection trigger API
